# House Price Prediction Using Linear Regression

## Project Overview

This project focuses on predicting house prices using the Linear Regression algorithm.

The objective is to build a machine learning model capable of estimating house prices based on various housing features such as area, bedrooms, bathrooms, stories, parking facilities, furnishing status, and other property characteristics.

The project demonstrates the complete machine learning workflow from data preprocessing to model evaluation and visualization.

---

## Problem Statement

House price estimation is an important application of machine learning in the real estate industry.

The goal of this project is to develop a regression model that can accurately predict housing prices using available property features.

---

## Dataset Information

The dataset contains housing-related attributes including:

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status
- Price (Target Variable)
